WorkoutApp

A React Native application designed to help users track their fitness journey, calculate health metrics, and discover healthy, goal-oriented nutritional plans and recipes.

Features

Workout Management: Add new exercises (AddExercises.js) and view your workout history (WorkoutHistory.js).

Fitness Tools: Built-in stopwatch (Stopwatch.js) and step counter (StepCounter.js) for active tracking.

Health Metrics: BMI Calculator (BMICalculator.js) to help monitor body composition.

Dietary Plans: Comprehensive guides on the food pyramid (PiramidaAlimentelor.js), diet plans (PlanuriAlimentare.js), and maintaining weight (PentruMentinere.js).

Healthy Recipes: Dedicated sections for vegan (ReteteVegane.js), high-protein (ReteteProteice.js), and weight-loss recipes (RetetePentruSlabit.js).

Getting Started

Prerequisites

Node.js installed on your machine.
React Native CLI or Expo CLI (the project configuration uses an app.json).

Installation

    Clone the repository to your local machine.

    Navigate into the project directory.

    Run npm install to install all necessary dependencies (as defined in the provided package.json and package-lock.json files).

    Run npm start to boot up the bundler and launch the app on your emulator or physical device.
